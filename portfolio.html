Portafolio

  • Email Marketing: Es la plataforma más grande en la que he trabajado en los últimos 2 años. Es una plataforma de Email Marketing(Envío de correos masivos), en la cual el usuario tiene a disposición estadísticas y muchas funcionalidades automáticas, ya que se desarrolló para que fuera inteligente. Esta plataforma fue y continua desarrollandose con PHP y un framework llamado Phalconphp, uno de los más rápidos en ejecución, también se usó Html, Javascript con Ember.js, Jquery, CSS y LESS, Php-ZMQ(Conexión por sockets), MySQL y muchas otras tecnologías.
  • Track: Es una pequeña plataforma que consume información de una base de datos MySQL y los transforma en gráficos de estadísticas usando Javascript. Esta información son registros de visitas con ubicación geografica que los usuarios proporcionan por medio de una app móvil android. Esta plataforma está desarrollada con PHP.
  • Track Android App: Esta aplicación móvil complementa a la plataforma track. Los usuarios se registran en la aplicación y esta toma la ubicación geografica en tiempo real. Esta desarrollada nativamente con JAVA y XML y consume la información necesaria de un API rest PHP.
  • Statistics Reporter: Esta plataforma desarrollada en PHP se alimenta de datos que proporcina un base de datos FIREBIRD y los transforma en estadisticas con JAVASCRIPT.
